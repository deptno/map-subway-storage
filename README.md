# 지하철 물품보관함 위치

> 아프지마 무릎 😭

![](public/screenshot.png)

지하철 역사의 물품보관함(해피박스) 위치를 지도와 함께 본다

## 로컬 배포
```shell script
npm run build # 데이터 빌드
npm run build.data
npm run publish
```
